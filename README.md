## 导线类

两个铜粒即可合成铜导线，再加一个塑料合成绝缘铜导线。
导线的电阻视材料而定，材料包含铜、铁、银、金、铝。

你也可以把染料放到工作台里面以合成不同颜色的导线，不同颜色的导线无法相互连接，需要依赖 双向/四向连接器 连接在一起。

你可以手持 机械动力 模组的 扳手 来改变导线的形状，这样可以很方便的理线，构建完美的电路网络。

同时，三根普通的相同颜色的绝缘导线可以合成小型电缆，它用于中距离输电（50m ~ 300m）；六根则可以合成中型电缆，它用于中远距离输电（300m ~ 1km）；九根可以合成大型电缆，它用于远距离和超远距离输电（1km ~ 3km）。

不同材料导线的电阻率和电阻图表如下：

| 材料 | 电阻率（固定） | 相对导电率 | 导线电阻 Ω/km |
|------|---------|------------------|------------|
| 银   | 0.0159  | 105.00%          | 15.9       |
| 铜   | 0.0172  | 100.00%          | 17.2       |
| 金   | 0.0244  | 70.50%           | 24.4       |
| 铝   | 0.0283  | 60.78%           | 28.3       |
| 铁   | 0.0978  | 17.60%           | 97.8       |

导电能力：银 > 铜 > 金 > 铝 > 铁

因此，远距离输电请考虑使用铝电缆。

如果你踩在裸露的导线上，36V ~ 72V 每秒扣除 1 颗生命值；72V ~ 144V 每秒扣除 2 颗生命值；144V ~ 288V 每秒扣除 4 颗健康值，获得 缓慢 debuff；288V 以上 2 秒内死亡。

### 合成配方

#### 金属粒

各金属粒均可通过工作台将对应金属锭切割合成，1个金属锭 → 9个金属粒，或通过机械动力模组的研磨机、切割机获得。

> 支持金属：铜、铁、银、金、铝

#### 裸导线（工作台）

```
[金属粒] [金属粒]
```
产出：2根 对应材料裸导线（铜/铁/银/金/铝）

```
[裸导线]
[塑料片]
```

> 塑料片来源：机械动力模组压榨机压榨橡胶原液，或熔炉熔炼沙砾+石油产物。

#### 彩色绝缘导线（工作台）

[绝缘导线] [任意染料]

```

[绝缘导线] [绝缘导线] [绝缘导线]

> 💡 提示：电缆可使用不同材料的绝缘导线合成，最终电阻取所用导线的加权平均值。远距离输电推荐铝制电缆，性价比最高。

## 变压器类

变压器包含升压变压器和降压变压器，两种变压器分为两类：小型和大型。

小型变压器用作低压，最高不超过 300V。

大型变压器用作高压，它可以把低压（300V 以内）升到高压和超高压，高压在 1kV ~ 5kV，超高压在 5kV ~ 50kV。

当然，遵循能量守恒，其电流会相应降低。

### 变压器规格说明

| 变压器类型      | 输入范围       | 输出范围        | 最大功率  |
|------------|------------|-------------|-------|
| 小型升压变压器    | 12V ~ 48V  | 48V ~ 300V  | 2kW   |
| 小型降压变压器    | 48V ~ 300V | 12V ~ 48V   | 2kW   |
| 大型升压变压器    | 48V ~ 300V | 1kV ~ 50kV  | 500kW |
| 大型降压变压器    | 1kV ~ 50kV | 48V ~ 300V  | 500kW |

### 合成配方

#### 铁芯（工作台）

```
    [铁锭]
[铁锭]    [铁锭]
    [铁锭]
```
产出：1个 铁芯

> 铁芯是变压器、电动机和发电机的核心部件。

#### 漆包铜线圈（工作台）

```
[绝缘铜导线] [绝缘铜导线] [绝缘铜导线]
[绝缘铜导线] [铁芯]       [绝缘铜导线]
[绝缘铜导线] [绝缘铜导线] [绝缘铜导线]
```
产出：1个 铜线圈

#### 小型升压变压器（工作台）

```
[铜线圈] [铁芯]   [铜线圈]
[铜导线] [铁锭]   [铜导线]
[铜线圈] [铁芯]   [铜线圈]
```
产出：1个 小型升压变压器

#### 小型降压变压器（工作台）

小型降压变压器与小型升压变压器配方相同，但放置方向（输入/输出端口）不同。合成后使用扳手右键可切换升压/降压模式。

产出：1个 小型降压变压器

#### 大型升压变压器（工作台）

```
[铜线圈]   [铁芯]   [铜线圈]
[铁芯]     [铜线圈] [铁芯]
[铜线圈]   [铁芯]   [铜线圈]
```
产出：1个 大型升压变压器

#### 大型降压变压器（工作台）

配方同大型升压变压器，使用扳手切换模式。

产出：1个 大型降压变压器

> ⚠️ 注意：超过变压器最大功率会导致变压器过热，长时间过载将烧毁变压器并引发火灾。

## 电池类

电池用于储存电能，适合在发电不稳定时作为缓冲，也可作为移动电源为便携设备供电。

### 电池规格说明

| 电池类型   | 容量       | 额定电压  | 最大充放电功率 | 化学体系  |
|--------|----------|-------|---------|-------|
| 铅酸蓄电池  | 10kWh    | 12V   | 1kW     | 铅酸   |
| 锂离子电池组 | 100kWh   | 48V   | 10kW    | 锂离子  |
| 大型储能电池 | 1,000kWh | 300V  | 100kW   | 固态锂  |

### 所需材料

#### 铅板（熔炉/铸造机）

将铅锭（可从铁矿石中少量提炼，或专属铅矿获取）放入铸造机。

```
铸造机：[铅锭] → [铅板] x2
```

#### 电解液瓶（工作台）

```
    [玻璃瓶]
[硫磺] [水桶] [硫磺]
    [玻璃瓶]
```
产出：2瓶 电解液（稀硫酸）

> ⚠️ 警告：电解液为腐蚀性液体，放入背包会造成持续伤害，请使用储物箱或玻璃容器。

#### 铅酸蓄电池（工作台）

```
[铅板]    [电解液瓶] [铅板]
[铅板]    [电解液瓶] [铅板]
[铜导线]  [铁框架]  [铜导线]
```
产出：1个 铅酸蓄电池

#### 锂电芯（电路合成终端）

```
输入：[铝板] x2 + [锂粉] x4 + [铜粉] x2 + [隔膜] x1
```
产出：2个 锂电芯

> 锂粉：通过研磨机研磨锂矿石获得。隔膜：塑料片 + 铜导线合成。

#### 锂离子电池组（工作台）

```
[铝板]    [锂电芯]  [铝板]
[锂电芯]  [电路板]  [锂电芯]
[铝板]    [铜导线]  [铝板]
```
产出：1个 锂离子电池组

#### 大型储能电池（工作台，需3x3完整格）

```
[铝板]      [锂离子电池组] [铝板]
[锂离子电池组] [高级电路板]  [锂离子电池组]
[铝板]      [铜导线x2]   [铝板]
```
产出：1个 大型储能电池

## 基础电子元件类

基础电子元件是构建各类电路设备的核心材料，部分元件需要在 **电路合成终端** 中合成。

### 材料预处理

#### 硅片（熔炉）

```
[沙子] x4 → [硅锭]（熔炉，需木炭x2）
[硅锭] → [硅片] x4（切割机）
```

#### 铜粉（研磨机）

```
[铜锭] → [铜粉] x2
```

#### 基础电路板（工作台）

```
[铜粉]  [硅片]  [铜粉]
[铜导线][木板]  [铜导线]
[铜粉]  [硅片]  [铜粉]
```
产出：1个 基础电路板

#### 高级电路板（电路合成终端）

```
输入：[基础电路板] x1 + [金粉] x2 + [硅片] x4 + [绝缘铜导线] x2
```
产出：1个 高级电路板

### 基础元件

#### 电阻（工作台）

```
[铁粉] [陶瓷] [铁粉]
```
产出：4个 电阻

> 电阻用于限流保护电路，可通过工作台调节阻值（右键使用扳手）。

#### 电容（工作台）

```
[铝箔] [电解液瓶] [铝箔]
       [陶瓷]
```
产出：2个 电容

> 铝箔：铝板放入压延机获得。

#### 电感（工作台）

```
    [铁芯]
[铜导线] [铜导线]
    [铁芯]
```
产出：2个 电感

#### 二极管（电路合成终端）

```
输入：[硅片] x2 + [铜粉] x1 + [铁粉] x1
```
产出：4个 二极管

#### 三极管（电路合成终端）

```
输入：[硅片] x3 + [金粉] x1 + [铜粉] x1
```
产出：2个 三极管

#### 集成运算放大器（电路合成终端）

```
输入：[高级电路板] x1 + [三极管] x4 + [电阻] x4 + [电容] x2
```
产出：1个 集成运算放大器

#### 晶振（电路合成终端）

```
输入：[石英] x2 + [铜粉] x1 + [银粉] x1
```
产出：2个 晶振

> 晶振用于数字电路时钟信号，频率可在放置后通过右键菜单调节（1Hz ~ 1MHz）。

#### 继电器（工作台）

```
[铜导线] [电磁铁] [铜导线]
[电阻]   [基础电路板] [电阻]
[铜导线] [弹簧片] [铜导线]
```
产出：1个 继电器

> 电磁铁：铁芯 + 绝缘铜导线 x4 工作台合成。弹簧片：铁锭放入压延机。

## 发电机以及电动机类

### 发电机

发电机将机械能（转矩）转换为电能，需接入机械动力模组的旋转轴输入。

#### 小型直流发电机（工作台）

额定输出：12V ~ 48V，最大功率：500W

```
[铁框架]  [铁芯]    [铁框架]
[铜线圈]  [永磁铁]  [铜线圈]
[铁框架]  [铜导线]  [铁框架]
```
产出：1个 小型直流发电机

> 永磁铁：铁锭 x2 + 磁石粉（研磨磁铁矿）放入熔炉烧制。

#### 中型交流发电机（工作台）

额定输出：120V / 230V 交流，最大功率：10kW

```
[铁芯]    [铜线圈]  [铁芯]
[铜线圈]  [永磁铁]  [铜线圈]
[铁芯]    [铜线圈]  [铁芯]
```
产出：1个 中型交流发电机

#### 大型工业发电机（电路合成终端）

额定输出：300V ~ 1kV，最大功率：1MW

```
输入：[中型交流发电机] x2 + [高级电路板] x2 + [大型铁芯] x2 + [铜线圈] x4
```
产出：1个 大型工业发电机

> 大型铁芯：铁芯 x4 在工作台 2x2 合成。

### 电动机

电动机将电能转换为机械能（转矩），可接入机械动力模组的旋转轴输出。

#### 小型直流电动机（工作台）

额定输入：12V ~ 48V，最大功率：250W

```
[铁框架]  [铜线圈]  [铁框架]
[铜线圈]  [铁芯]    [铜线圈]
[铁框架]  [铜导线]  [铁框架]
```
产出：1个 小型直流电动机

#### 中型交流电动机（工作台）

额定输入：120V / 230V 交流，最大功率：5kW

```
[铁芯]    [铜线圈]  [铁芯]
[铜线圈]  [铁芯]    [铜线圈]
[铜导线]  [铁框架]  [铜导线]
```
产出：1个 中型交流电动机

#### 变频驱动器（电路合成终端）

用于调节交流电动机转速，支持 0% ~ 150% 额定转速调节。

```
输入：[高级电路板] x2 + [集成运算放大器] x2 + [二极管] x8 + [电容] x4 + [铝板] x2
```
产出：1个 变频驱动器

> 💡 提示：变频驱动器 + 中型/大型交流电动机 可实现无级调速，对需要精确控制转速的机器非常有用。

## 连接器类

连接器用于管理和分配电路网络，解决不同颜色导线互联、多路输出等需求。

### 单向/双向连接器

#### 导线接头（工作台）

用于将不同颜色的导线临时对接。

```
[任意颜色绝缘导线] [铜粒] [任意颜色绝缘导线]
```
产出：1个 导线接头

#### 双向连接器（工作台）

允许两条不同颜色的导线互联。

```
    [铜粒]
[绝缘导线] [铜板] [绝缘导线]
    [铜粒]
```
产出：1个 双向连接器

> 铜板：2个铜锭工作台合成。

#### 四向连接器（工作台）

允许四个方向的导线（可不同颜色）互联汇流。

```
[铜粒]  [绝缘导线] [铜粒]
[绝缘导线] [铜板]  [绝缘导线]
[铜粒]  [绝缘导线] [铜粒]
```
产出：1个 四向连接器

### 分线器与汇流排

#### 1分4分线器（工作台）

将一路电源分成四路独立输出。

```
[铜导线] [铜板]   [铜导线]
[铜导线] [基础电路板] [铜导线]
[铜粒]   [铜粒]   [铜粒]
```
产出：1个 1分4分线器

#### 汇流排（工作台）

多路电源的汇聚节点，最多支持8路输入/输出。

```
[铜板]   [铜板]   [铜板]
[铜导线] [铁框架] [铜导线]
[铜板]   [铜板]   [铜板]
```
产出：1个 汇流排

### 断路器与保险丝

#### 保险丝（工作台）

当电流超过额定值时熔断保护线路，需要更换。

```
[铜粒] [铁粉] [铜粒]
```
产出：4个 保险丝（额定电流 5A）

> 更高电流的保险丝可使用银粉/金粉替代铁粉，额定电流分别为 20A / 50A。

#### 断路器（工作台）

可重置的过流保护器件，过载后手动右键复位。

```
[铁锭]  [电磁铁] [铁锭]
[弹簧片] [基础电路板] [弹簧片]
[铜导线] [铁框架] [铜导线]
```
产出：1个 断路器（额定电流 20A）

#### 漏电保护断路器（电路合成终端）

同时具备过流保护和漏电保护功能。

```
输入：[断路器] x1 + [集成运算放大器] x1 + [三极管] x2 + [电阻] x2
```
产出：1个 漏电保护断路器

## 数字电路逻辑门类

数字电路逻辑门基于标准逻辑电平（低电平 0V，高电平 5V/3.3V），可与红石信号互转，适合构建复杂的自动化控制逻辑。

> 所有逻辑门均在 **电路合成终端** 中合成。

### 基本逻辑门

#### NOT 门（非门）（电路合成终端）

```
输入：[基础电路板] x1 + [三极管] x1 + [电阻] x2
```
产出：2个 NOT 门

#### AND 门（与门）（电路合成终端）

```
输入：[基础电路板] x1 + [三极管] x2 + [电阻] x3
```
产出：2个 AND 门

#### OR 门（或门）（电路合成终端）

```
输入：[基础电路板] x1 + [三极管] x2 + [电阻] x2 + [二极管] x2
```
产出：2个 OR 门

#### NAND 门（与非门）（电路合成终端）

```

### 信号转换器

#### 红石→数字信号转换器（工作台）

将红石信号（强弱）转换为数字逻辑电平（0/1）。

```
[红石粉]  [硅片]    [铜导线]
[电阻]    [基础电路板] [电阻]
[铜导线]  [铜粒]    [红石粉]
```
产出：1个 红石→数字转换器

#### 数字信号→红石转换器（工作台）

将数字逻辑信号输出为红石信号。

```
[铜导线]  [硅片]    [红石粉]
[电阻]    [基础电路板] [电阻]
[红石粉]  [铜粒]    [铜导线]
```
产出：1个 数字→红石转换器

#### 模拟数字转换器 ADC（电路合成终端）

将模拟电压信号（0V ~ 5V）转换为 8 位数字信号输出。

```
输入：[高级电路板] x1 + [集成运算放大器] x2 + [电容] x4 + [电阻] x8 + [晶振] x1
```
产出：1个 ADC 模块

#### 数字模拟转换器 DAC（电路合成终端）

将数字信号转换为对应模拟电压。

```
输入：[高级电路板] x1 + [集成运算放大器] x2 + [电阻] x8 + [电容] x2
```
产出：1个 DAC 模块

        [硅片]
```

#### LED 灯条（工作台）

```
[LED灯] [LED灯] [LED灯]
[铜导线] [铝板] [铜导线]
```
产出：1个 LED 灯条（长度 1格，可串联延伸）

#### 荧光灯管（工作台）

```

> 汞粉：研磨辰砂矿石获得，有毒，操作时请佩戴防护手套（游戏内橡胶手套）。荧光粉：骨粉 + 萤石粉合成。

### 电路合成终端

电路合成终端是合成高级电子元件的核心设备，需要持续供电（12V ~ 48V，至少 50W）才能工作。

#### 电路合成终端（工作台）

```
[铁板]    [高级电路板] [铁板]
[基础电路板] [玻璃板]  [基础电路板]
[铁框架]  [铜导线]   [铁框架]
```
产出：1个 电路合成终端

> 铁板：铁锭 x2 在工作台合成（2x1）。首次使用需要连接电源后右键开机。

### 防护装备

#### 绝缘手套（工作台）

佩戴后可安全触碰 288V 以下的裸露导线，不受电击伤害。

```
[橡胶] [橡胶] [橡胶]
[橡胶]        [橡胶]
```
产出：1双 绝缘手套

> 橡胶：机械动力模组压榨树液获得，或橡胶树木材压榨。

#### 绝缘靴（工作台）

穿戴后踩在 144V 以下裸露导线上不受伤害，但高于 144V 仍有危险。

```
[橡胶]        [橡胶]
[橡胶] [橡胶] [橡胶]
[橡胶]        [橡胶]
```
产出：1双 绝缘靴

#### 完整绝缘套装效果

同时穿戴绝缘手套 + 绝缘靴时，可完全免疫 1kV 以下的电击伤害，1kV 以上仍会受到 50% 伤害。
