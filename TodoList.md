## 导线类

### 1. 裸导线 (Bare Wire) — 5种材料：铜/铁/银/金/铝

- **状态**: 🟡 部分完成（程序实现 90%，模型 0%）
- **优先级**: 🔴 高
- **简介**: 两根金属粒合成的基础导线，电阻最低但无绝缘。踩踏会受到电击伤害（需防护装备）。5种材料导电率各异。

#### 建模任务

- [ ] 3D模型设计（BlockBench）
  - [ ] 直线段建模
  - [ ] L形、T形弯曲建模
  - [ ] 十字交叉建模
  - [ ] 纹理和材质设计（金属光泽）
  - [ ] 碰撞箱设置（当前代码已设 6px 宽薄片）
- [ ] 5种材料颜色变体纹理（铜/铁/银/金/铝）
- [ ] 模型文件格式转换与优化

#### 程序实现

- [x] 物品注册 — BareWireItem extends BlockItem，注册名 bare_{material}_wire
  - [x] 设置基础属性（maxStackSize=64）
  - [x] Tooltip 显示材料/电阻率/导电率
- [x] Block实现（BareWireBlock）
  - [x] 6向 BooleanProperty 连接（NORTH/EAST/SOUTH/WEST/UP/DOWN）
  - [x] 裸导线互联 + 不连绝缘导线
  - [x] 实体踩踏检测（stepOn）— 电击伤害计算（36V~288V+ 分级）
  - [x] getShape() 返回薄片碰撞箱
- [x] BlockEntity（WireBlockEntity）
  - [x] 电压存储变量
  - [x] 能量存储（EnergyStorage 1000FE）
  - [x] 每 5 tick 向邻居传播能量
  - [x] NBT保存/读取（saveAdditional / loadAdditional）
- [x] 网络系统（WireNetwork + WireNetworkManager）
  - [x] BFS 导线网络发现
  - [x] 自动网络形成
  - [x] 能量均衡分配算法
- [ ] 扳手交互（右键旋转导线方向）
- [ ] Renderer实现（TESR）
  - [ ] 动态纹理渲染
  - [ ] 电流动画效果

#### 合成配方

- [x] 铁裸导线: 2铁粒 → 2裸铁导线
